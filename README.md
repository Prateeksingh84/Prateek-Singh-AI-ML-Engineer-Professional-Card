## 👋 About

A clean, responsive, and professional **Astro-based portfolio card** for **Prateek Singh**, designed to showcase an AI/ML Engineer profile with resume, GitHub, and LinkedIn links.

This project is a minimal personal portfolio card built using **Astro**, HTML, and CSS. It presents a professional AI/ML Engineer profile with a modern gradient card UI, profile image, skill chips, and quick-access social links.

## 📦 Installation

**1. Clone the repository**
```bash
git clone https://github.com/Prateeksingh84/Prateek-Singh-AI-ML-Engineer-Professional-Card.git
```

**2. Move into the project folder**
```bash
cd Prateek-Singh-AI-ML-Engineer-Professional-Card
```

**3. Install dependencies**
```bash
npm install
```

If PowerShell blocks npm, use:
```powershell
npm.cmd install
```

---

## ▶️ Run Locally

```bash
npm run dev
```

For PowerShell:
```powershell
npm.cmd run dev
```

Open in browser:
```text
http://localhost:4321
```

## 🏗️ Build for Production

For PowerShell:

### 🌐 Deploy on Vercel

**1. Push code to GitHub**
```bash
git add .
git commit -m "feat: add professional AI/ML engineer portfolio card"
git push origin main
```

**2. Import repository on Vercel**
- Go to Vercel
- Click **Add New Project**
- Import your GitHub repository
- Keep default Astro settings
- Click **Deploy**

### 🌐 Deploy on Netlify

**Build command**
```bash
npm run build
```

**Publish directory**
```text
dist
```
